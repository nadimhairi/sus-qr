# Sus QR

Sus QR is a community-powered fraud registry for DuitNow QR payments in Malaysia. Before paying, Sus QR will extract data and instantly check if other people have reported scanned QR code and return the fraud score.

### 1. Install the packages

```bash
npm install
```

### 2. Set the .env in ./sus-qr-api

```bash
DATABASE_URL: example
```

### 3. Run the api server

```bash
npm run dev:api
```

### 4. Start the app

```bash
npm run start:app
```
